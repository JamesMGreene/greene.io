    • Collaborator
    • Official jQuery Team Member, "Testing" sub-team
    • Addon Creator
      • qunit-assert-html:
        A QUnit addon for assertion methods to compare two HTML strings for equality after a normalization process.

        A QUnit addon for reporting test results in the XUnit format. Great for integration with Jenkins CI servers.

  • jQuery — jQuery is a fast, small, and feature-rich JavaScript library. (Heard of it? )
    • Official jQuery Team Member, "Testing" sub-team
    • Plugin Creator
      • jquery.outerHtml:
        A simple jQuery plugin to get the outer HTML of an element, a la IE's [formerly] proprietary outerHTML property on elements.
          → Also available on the jQuery Plugin Registry.
      • jquery.hub:
        A well-tested jQuery plugin implementing enterprise-level messaging patterns atop the OpenAjax [Unmanaged] Hub. Great for decoupling cross-component communication on the client-side.
          → Also available on the jQuery Plugin Registry.
      • jquery.textSelect:
        jQuery plugin that creates a bindable special event for user text selection: "textSelect".
          → Also available on the jQuery Plugin Registry.
      • jquery.multiclick:
        jQuery plugin that creates a bindable special event for a configurable number of clicks: "multiclick".
          → Also available on the jQuery Plugin Registry.
  • Node.js — Node.js is server-side JavaScript execution environment with an event-driven, non-blocking I/O model.
    • Module Creator
      • gc2gh-issue-migrator:
        Migration tool(s) for fetching issues from the Google Code Issue Tracker API and translating that data into a GitHub Issue Tracker offline-importable format.
          → Also available on the NPM Registry.
      • gcph-client:
        A Node.js client for the Google Code Project Hosting Issue Tracker API.
          → Also available on the NPM Registry.
      • flex-sdk:
        A Node.js module to download and "install" the latest open sourced Apache/Adobe Flex SDK.
          → Also available on the NPM Registry.
      • dos2unix:
        A Node.js module to convert text files with DOS line breaks to UNIX line breaks, i.e. like using dos2unix.
          → Also available on the NPM Registry.
  • Grunt — "The JavaScript Task Runner"
    • Plugin Creator
      • grunt-chmod:
        A Grunt task plugin to modify file permissions, i.e. like using chmod.
          → Also available on the NPM Registry.
          → Also available on the Grunt Plugin Registry.
      • grunt-mxmlc:
        A Grunt task plugin to compile Adobe Flex/ActionScript/MXML/FLV/etc. apps with Adobe's mxmlc compiler.
          → A work in progress... resolving some unexpected cross-platform compatibility issues presently.
  • jWalker — jWalker is a well-tested cross-browser JavaScript implementation of the TreeWalker class defined in the W3C specification for DOM Traversal (DOM Level 2).
